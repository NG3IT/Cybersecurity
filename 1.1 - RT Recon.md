## Whois

```bash
$ whois <domain>
$ whois thmredteam.com
```

```bash
$ nslookup <site>
$ nslookup cafe.thmredteam.com
```

```bash
$ dig <site>
$ dig cafe.thmredteam.com
```

```bash
$ host <site>
$ host cafe.thmredteam.com
```

```bash
$ traceroute <site>
$ traceroute cafe.thmredteam.com
```
